注解

Sep 19, 2019

注解语法

public @interface AnnotationName{

}

  • 所有的注解都隐式地扩展自java.lang.Annotation接口；
  • 这个接口是一个常规接口不是注解接口；
  • 注解元素的类型:
    • 基本类型
    • String
    • Class
    • enum类型
    • 注解类型
    • 由前面类型组成的数组

注解

@AnnotationName(elementName=value…)

  • 一个注解元素永远不能设为null，默认值也不能为null
  • 在注解中引入循环依赖是一种错误

注解各类声明

  • 声明用法可以出现在：
    • 包
    • 类（包括Enum）
    • 接口（包括注解接口）
    • 方法
    • 构造器
    • 实例域 （包含enum常量）
    • 局部变量
    • 参数变量
    • 类型参数
  • 对局部变量的变量只能在源码级别上进行处理。类文件并不描述局部变量。因此所有的局部变量注解在编译完一个类的时候就会被遗弃掉。同样的，对包的注解不能在源码级别之外

注解类型用法

  • 类型用法可以出现在：
    • 与泛化类型引元一起使用
    • 数组中的任何位置
    • 与超类和实现接口一起使用
    • 与构造器一起使用
    • 与强制类型转换和instanceof检查一起使用
    • 与异常违约一起使用
    • 与通配符和类型边界一起使用
    • 与方法和构造器引用一起使用

标准注解

注解接口 应用场合 目的
Deprecated 全部 将项标记为过时的
SuppressWarnings 除了包和注解之外的所有情况 阻止给某个给定类型的警告信息
SafeVarargs 方法构造器 断言varargs
Override 方法 检查方法是否覆盖了某一个超类方法
FuntionalInterface 接口 将接口标记为只有一个抽象方法的函数式接口
PostConstruct/PreDestroy 方法 被标记的方法应该在构造之后或移除之前立即被调用
Resource 类、接口、方法、域 在类或接口上：标记为在其他地方要用的资源；在方法或域上，为注入做标记
Resources 类、接口 一个资源数组
Generated 全部
Target 注解 指明可以应用这个注解的那些项
Retention 注解 指明这个注解可以保留多久
Documented 注解 指明这个注解应该包含在注解项的文档中
Inheited 注解 指明当这个注解应用于某一个类的时候，能够自动被它的子类继承
Repeatable 注解 指明这个注解可以在同一个项上应用多次

用于编译的注解

  • @Deprecated
  • @SupressWarnings
  • @Generated 注解的目的是供代码生成工具来使用。任何生成的代码都可以被注解，从而与程序员提供的代码区分开来

用于管理资源的注解

  • @PostConstruct
  • @PreDestroy
  • @Resource

元注解

  • @Target 注解可以应用的项
  • @Retention
  • @Documented 用于将使用该注解项纳入文档
  • @Inherited 只能应用于对类的注解。如果一个类具有继承注解，那么它的所有子类都自动具有同样的注解

@Target(ElementType.?)
|元素类型|注解适合场合|
|——-|———–|
|ANNOTATION_TYPE|注解类型声明|
|PACKAGE|包|
|TYPE|类（包括enum）及接口（包括注解类型）|
|METHOD|方法|
|CONSTRUCTOR|构造器|
|FIELD|成员域（包括enum常量）|
|PARAMETER|方法或者构造器|
|LOCAL_VARIBLE|局部变量|
|TYPE_PARAMETER|类型参数|
|TYPE_USE|类型用法|

@Retention(RententionPolicy.?)
|保留规则|描述|
|——-|—-|
|SOURCE|不包括在类文件中的注解|
|CLASS|包括在类文件中的注解，但是虚拟机不需要将他们载入|
|RUNTIME|包括在类文件中的注解，并且由虚拟机载入。通过反射API可以获得他们|

@Repeatable()

  • 处理可重复注解时必须非常仔细。如果调用getAnnotation来查找某个可重复注解，而该注解又确实重复了，那么就会null。这是因为重复注解被包装到了容器注解中
  • 在这种情况下应该调用getAnnotationType，这个调用会遍历容器并给出一个重复注解的数组。如果只有一条注解，那么该数组的长度为1.
